# Step 5: Process ESJC Data

## Intent
Load and process the eSJC (Electronic Supplementary Job Card) Detailed Report
for the SQ A350 fleet. This is an independent track from the SAP processing
(Steps 1-4). Produces a cleaned and restructured ESJC output.

## Input
- eSJC Detailed Report Excel file (e.g., `eSJC_DetailedReport A350 FLEET 01-31 JAN 2026.xlsx`)
- Expected columns include: ESJC no., A/C Reg, DATE(COMPLETE), DATE(RAISED),
  ARR FLT, CHECK, LAE SIGN OFF LIC/AUTH No., JOB No.,
  ORIGINATION CARD CROSS REFERENCE, SNO, DEFECT/ACTION REQUIRED,
  ACTION TAKEN, MHR DECIMAL, MM OR RELEVENT APPROVED INSTRUCTION REFERENCE

## Processing Rules

### 1. Column Selection
Select exactly these 14 columns:
- ESJC no.
- A/C Reg
- DATE(COMPLETE)
- DATE(RAISED)
- ARR FLT
- CHECK
- LAE SIGN OFF LIC/AUTH No.
- JOB No.
- ORIGINATION CARD CROSS REFERENCE
- SNO
- DEFECT/ACTION REQUIRED
- ACTION TAKEN
- MHR DECIMAL
- MM OR RELEVENT APPROVED INSTRUCTION REFERENCE

### 2. Add From/To Columns
- Add `From` column with empty string for all rows
- Add `To` column with value `SIN` for all rows

### 3. Merge Columns
Create two new merged text columns:
- **Defect/Action required**: Concatenate (space-joined, NaN as empty):
  `JOB No.` + `ORIGINATION CARD CROSS REFERENCE` + `DEFECT/ACTION REQUIRED` +
  `MM OR RELEVENT APPROVED INSTRUCTION REFERENCE`
- **Corrective Action**: Concatenate (space-joined, NaN as empty):
  `MM OR RELEVENT APPROVED INSTRUCTION REFERENCE` + `ACTION TAKEN`

### 4. Drop All-NaN Rows
Remove any row where ALL columns are NaN.

### 5. Forward-Fill Metadata
For rows where the first 9 columns (ESJC no. through ORIGINATION CARD CROSS REFERENCE)
are ALL NaN (these are sub-rows of a parent ESJC entry):
- Forward-fill only those 9 columns from the preceding row
- Only fill rows that match the condition (all 9 are NaN), not arbitrary NaN values

### 6. Rearrange to Final Order
Reorder columns to:
1. ESJC no.
2. A/C Reg
3. DATE(COMPLETE)
4. DATE(RAISED)
5. ARR FLT
6. CHECK
7. LAE SIGN OFF LIC/AUTH No.
8. JOB No.
9. ORIGINATION CARD CROSS REFERENCE
10. SNO
11. DEFECT/ACTION REQUIRED
12. Defect/Action required (merged)
13. MHR DECIMAL
14. MM OR RELEVENT APPROVED INSTRUCTION REFERENCE
15. From
16. To
17. ACTION TAKEN
18. Corrective Action (merged)

## Output
Cleaned and restructured ESJC DataFrame with 18 columns.

## Review Criteria
- No rows where all columns are NaN
- From column should be all empty strings
- To column should be all "SIN"
- Merged columns should contain concatenated text from source columns
- Forward-fill should only apply to sub-rows (where first 9 cols are all NaN)
