# Step 3: Classify SAP Defects

## Intent
Categorize each defect row into one of five categories based on keyword matching
in the Defect and Corrective Action columns. Categories help prioritize and
route maintenance actions.

## Input
Expanded SAP DataFrame from Step 2 with columns including:
Defect, Corrective Action, and all columns from previous steps.

## Processing Rules

### 1. Oxygen Issue Mask
A row is classified as "Oxygen Issue" when:
- **Defect** contains ANY of: `OXY`, `o2`, `Bottle`, `BTL`, `SOK` (case-insensitive)
- **AND Corrective Action** contains ANY of: `replace`, `rpl`, `service`, `uplift`,
  `raised`, `insp`, `inspection`, `OXY`, `o2`, `Bottle`, `BTL`, `SOK` (case-insensitive)

### 2. Toilet Choke Mask
A row is classified as "Toilet Choke" when:
- **Defect** contains ANY of: `toilet`, `choke` (case-insensitive)
- **AND Corrective Action** contains ANY of:
  `FOD AND WASTE WATER CLEARED`, `CHOKED CLEARED`, `TOILET BOWLS`,
  `FLUSHING`, `TOILET BOWL FOUND CLOGGED` (case-insensitive)

### 3. NIL DEFECT Mask
A row is classified as "NIL DEFECT" when:
- **Defect** starts with `NIL DEFECT` (case-insensitive, regex: `^NIL DEFECT`)
- **AND Corrective Action** contains ANY of:
  `CLOSURE NOTED AS NIL DEFECT`, `NOTED THANK`, `NIL NOTED`,
  `NIL FAULTED NOTED`, `CLOSURE NIL NOTED THANK`, `CLOSURE WELL NOTED`
  (case-insensitive)

### 4. Open Defect Mask
A row is classified as "Open Defect" when:
- **Defect** does NOT contain `defect action - closure` (case-insensitive)
- **AND Corrective Action** does NOT contain `defect action - closure` (case-insensitive)

### 5. Classification Priority
Apply masks in order using numpy.select:
1. Oxygen Issue (highest priority)
2. Toilet Choke
3. NIL DEFECT
4. Open Defect
5. Default: "Unclassified"

Store result in new column `Defect_Category`.

## Output
Same DataFrame with new `Defect_Category` column containing one of:
Oxygen Issue, Toilet Choke, NIL DEFECT, Open Defect, Unclassified.

## Review Criteria
- Every row should have a Defect_Category value (no nulls)
- Categories should be one of the 5 valid values
- Oxygen/Toilet/NIL masks should be mutually exclusive by priority order
- Open Defect is the catch-all for non-closure items
